▼

Pull from Zoho

▼
Type the Zoho SO # (e.g. SO-22815) or invoice # (e.g. INV-022496). Preview gives a quick summary — customer, totals, status, price-check verdict. Pull opens the review modal where you confirm the line items and apply them to the DIRECT table. Pending sheet mirrors all direct-channel Zoho SOs in real time; only the ones you Pull reach the working sheet.

Kit Expansion

▼
Select kit rows in All Orders → Expand Selected → opens a popup modal with the Sales Description reference, per-component exclusion, and a per-kit Spares input, then commits from there. Spares for us is additive: 0 ships exactly the customer's row qty; N adds N extra kits per row. READY kits (K-* aisle) and already-expanded kits are refused with a reason.

Price Audit

▼
Compares Zoho selling_price vs eBay currentPrice (MI) across all items. Surfaces drifts so admin can fix Zoho before quotes.
